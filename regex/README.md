# Regular Expression

| Regex | 意義 |
| ----- | ----- |
| `^word` | 字串行首 |
| `word$` | 字串行尾 |
| `.`     | 一個任意字元  |
| `\`     | 跳脫字符 |
| `*`     | 重複零個至多個的前一個字元 |
| `[list]`  | 想要擷取的字元 |
| `[n1-n2]` | 想要擷取的字元範圍 |
| `[^list]` | 列出不要的字串或範圍 |
| `\{n,m\}` | 連續 n 到 m 個的『前一個 RE 字符』 |
