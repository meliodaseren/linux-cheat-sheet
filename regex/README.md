# Regular Expression

| Regex | 意義 |
| ----- | ----- |
| `^word` | 字串行首 |
| `word$` | 字串行尾 |
| `.`     | 一個任意字元  |
| `\`     | 跳脫字符 |
| `*`     | 重複零個至多個的前一個字元 |
| `[list]`  |  |
| `[n1-n2]` |  |
| `[^list]` |  |
| `\{n,m\}` |  |
